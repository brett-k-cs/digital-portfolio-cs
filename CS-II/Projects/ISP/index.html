ISP - SchoolFood [Graded]

Description

Our business, SchoolFood, provides an application for schools in order to allow online ordering for students. It will have every option that each resteraunt has in order to make this experience as smooth as possible. This speeds up the time taken to get your lunch order. The application would also allow students to be productive while they wait for their food since they no longer have to wait in line. This means students could talk or do homework. We will have a backend server, a frontend website, and a mobile application. It will have it's own payment system as well as hooking into PayPams to make ordering easy without charging insane fees (SchoolPay!).
